买了新android手机，一定要做以下几件事

candy2026-04-292026-04-29

这份教程将带你从底层权限开始，一步步打造一个更私密、更高效的系统。

第一阶段：ADB 降维打击（清道夫模式）

很多系统自带的无用应用无法直接长按卸载，我们需要借助电脑端的 ADB (Android Debug Bridge) 工具。

  1. 开启开发者选项与 USB 调试

步骤：进入手机 设置 -> 关于手机 -> 连续点击 版本号 或 系统版本（如 MIUI/系统版本号）7次，直到提示“您已处于开发者模式”。

开启调试：返回 设置 -> 系统与更新（或更多设置） -> 开发者选项 -> 打开 USB 调试。

  1. 电脑端配置 ADB 并连接

下载工具：在电脑上下载 SDK Platform-Tools，解压到任意目录。

连接手机：用数据线连接电脑和手机，手机端弹出“是否允许 USB 调试”时，勾选“始终允许”并确认。

打开终端：在电脑解压的 Platform-Tools 文件夹地址栏输入 cmd 或 powershell 回车。输入 adb devices，如果出现一串字符加上 device 字样，说明连接成功。

  1. 查找并卸载无用应用

列出所有应用包名：输入命令 adb shell pm list packages （建议配合 grep 过滤，例如 adb shell pm list packages | findstr browser）。

免 Root 卸载命令：

adb shell pm uninstall -k –user 0 <应用包名>

(注：<应用包名> 是类似 com.tencent.soter.soterserver 的格式。–user 0 表示仅卸载当前用户的应用，不会损坏系统底层，恢复出厂设置即可还原。)

⚠️ 极客推荐工具：如果你觉得敲代码麻烦，强烈推荐在电脑上下载开源可视化工具 [ADB AppControl] 或 [Universal Android Debloater]。它们自带国内外常见手机的“安全卸载列表”，可以直接勾选一键卸载自带浏览器、应用商店、游戏中心等，防止误删变砖。

第二阶段：拥抱 Google 生态（洗白模式）

国内安卓环境生态恶劣，主流软件开屏广告泛滥。转移到 Google Play 下载应用是根治广告的最有效方式。

  1. 安装 VPN / 代理客户端

因为新手机没有 VPN 且打不开国外网站，你需要先通过电脑下载好客户端的 APK 安装包（如 Clash Meta for Android, v2rayNG 等），通过 USB 拷贝到手机安装。

配置好你的节点订阅，确保手机可以顺畅访问国际网络。

  1. 安装 Google Play 商店

内置框架激活：如今绝大多数国产旗舰机（小米、OV、荣耀等）其实都内置了 Google 基础服务框架（GMS）。

步骤：进入手机 设置 -> 搜索 Google 或 谷歌基础服务 -> 将其开关打开。

安装 Play Store：框架开启后，直接在手机自带的应用商店搜索“Google Play”，或者前往 APKMirror 网站下载最新版的 Google Play Store APK 安装即可。

(注：如果是华为等未内置 GMS 的设备，则需要借助 出境易、Gspace 或刷机/使用 microG 等复杂方案。)

  1. 应用下载策略

日常软件：微信、B站、网易云等尽量都在 Google Play 下载。Play 商店版通常受制于 Google 严格的隐私和后台规范，极少有开屏广告，且不会随意在后台互相唤醒。

开源软件：推荐安装 F-Droid（开源应用商店），从中下载没有任何追踪代码的开源工具软件。

第三阶段：深层隐私硬化（隐身模式）

这一步是关掉那些“看着就烦”和“偷偷看你”的功能。

1. 关闭系统级广告

  • 广告 ID 重置：设置 -> Google -> 广告 -> 重置广告 ID 并启用“选择退出广告个性化”。
  • DNS 加密：设置 -> 连接与共享 -> 私人 DNS。输入 dns.adguard.com 或 p2.freedns.kontur.io，可以从协议层拦截大量 App 内广告。

2. 斩断“触角”

  • 关闭快应用：在设置里搜索“快应用”，撤回其所有权限并关闭“检测链接”。
  • 关闭负一屏/信息流：桌面设置中直接关闭“负一屏”或“Google Feed”。
  • 关闭用户体验计划：搜索“加入用户体验计划”或“自动发送诊断数据”，全部关掉。

3. 应对监管与便利性建议 (进阶)

需求 建议操作 目的
应用防监控 安装 Shizuku + App Ops 深度管理权限，拒绝应用读取你的联系人、通话记录。
存储空间隔离 使用 Storage Isolation 强迫软件只能读写自己的文件夹，防止扫描你的相册。
躲避反诈检测 关掉内置垃圾清理/安全扫描 某些自带的安全中心会扫描已装 APK 并上报，建议关闭云端扫描。
极速响应 开发者选项 -> 动画缩放改为 0.5x 视觉感受提升一倍，手机瞬间变快。

4：进阶防监管与隐私保护（核心极客建议）

为了防止被过度监控、画像，以及避免某些流氓 App 扫描你的手机，建议采取以下措施：

  1. 更换安全的输入法（极度重要）

风险：国内主流输入法（搜狗、百度等）都有云端词库，你的所有输入记录（包括密码、聊天、搜索）都会被上传并可能被审查/用于广告画像。

对策：立即卸载系统自带输入法，换用 Gboard (Google 键盘)，并在 Gboard 设置中关闭“分享使用情况统计”。或者使用完全本地化的开源输入法 Rime (同文输入法) 或 FUTO Keyboard。

  1. 配置加密私密 DNS (DoT)

风险：默认情况下，你的运营商可以知道你访问了哪些网站（DNS劫持与污染）。

对策：设置 -> 网络和连接 -> 找到 私人 DNS (Private DNS) -> 选择“指定私密 DNS提供商”，填入 dns.google 或 1dot1dot1dot1.cloudflare-dns.com。这不仅能防运营商劫持，还能在一定程度上绕过轻度的 DNS 污染。

  1. 使用“工作空间”隔离毒瘤应用 (沙盒机制)

风险：微信、支付宝、淘宝等 App 需要过多权限，且经常在后台扫描手机文件或互相唤醒。

对策：利用安卓的 Work Profile (工作空间) 机制，将这些 App 关进“沙盒”。

下载开源应用 Shelter 或 Island (炼妖壶)。

它们会在手机里创建一个独立的“空间”，你可以把流氓软件克隆进去。

优点：工作空间与主空间的数据、文件、通讯录完全物理隔离（微信扫不到你相册里的敏感图）；当你不需要用它们时，可以一键冻结整个工作空间，彻底阻断它们在后台耗电和监控。

  1. 权限与剪贴板控制

最后的“绝杀”建议

如果你追求极致的清爽，可以尝试 “两套方案”：

  1. 工作生活分离：利用安卓自带的 “多用户” 或 “手机分身” 功能。一个空间放国产流氓软件（微信、支付宝等），另一个空间纯放 Google 生态和私人数据。
  2. 彻底跳出：如果你的手机支持，直接解开 Bootloader 锁，刷入 LineageOS 或 Pixel Experience 等类原生 ROM。这是物理意义上的“重生”，从源头上彻底切断了所有预装监控。

温馨提示：在卸载系统应用前，记得先备份重要数据。万一不小心删掉了系统桌面导致黑屏，可以通过 adb install 一个第三方桌面救回来。
